Contextual 嵌入增强（Anthropic 方案）
系统将为每个 Chunk 自动生成约 100 token 的上下文摘要前缀，存入 L0 摘要字段。检索时优先加载摘要，显著降低 Token 消耗，检索隔阈率降低 35~67%。
⚡
智能上下文模式 — 额外 LLM 调用
此模式使用 LLM 理解文档结构后再切片，每个文档将消耗额外 LLM 调用（约 500~2000 tokens/文档）。建议仅用于高价值、需精确检索的核心文档。
